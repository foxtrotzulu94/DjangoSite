Hey there, I'm Javier!

{# Short description/backstory #}

Welcome! I'm a Canadian Software Developer originally from Caracas, Venezuela. I started programming back in 2007 when I wanted to make my own videogame. Back then, I tried building a game with a Doom source port to give it a first try. Though I didn't get far, I did learn a great deal about videogame development that still holds true to this day. Later in 2010, I picked up Ubuntu Linux out of curiosity. After figuring out the terminal and customizing the distro, I wanted to learn more about operating systems and the hardware they run on. These two events led me down the path of Computer Engineering.

Since 2013, I've been programming non-stop! Working long hours, both with teams and alone, in hackathons, game jams, at school, and at home to make cool stuff.

This site can give you an idea of my experience, skills and interests. Go on and take a look!
